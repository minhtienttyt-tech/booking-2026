# Hệ thống Quản lý Booking 2026 - Nhà hàng Memory

Hệ thống quản lý đặt lịch chuyên nghiệp dành cho Nhà hàng/Khách sạn, giúp theo dõi các booking từ đại lý du lịch (EXO, Discova, Aurora...) và khách lẻ.

## Tính năng chính
- **Dashboard**: Tổng quan số lượng booking, khách (Pax) và doanh thu dự kiến.
- **Quản lý Booking**: Thêm, sửa, xóa các lịch đặt chỗ một cách trực quan.
- **Bộ lọc thông minh**: Lọc theo đại lý, trạng thái (Xác nhận, Chờ, Hủy).
- **Xuất dữ liệu**: Hỗ trợ xuất file Excel và PDF chuyên nghiệp.
- **Bảo mật**: Truy cập bằng mã PIN (Mặc định: **2026**).

## Cách sử dụng
1. Tải toàn bộ thư mục về máy tính.
2. Chạy file `run_booking.bat` để khởi động hệ thống.
3. Trình duyệt sẽ tự động mở tại địa chỉ: `http://localhost:3001`.
4. Nhập mã PIN **2026** để đăng nhập.

## Công nghệ sử dụng
- HTML5 / CSS3 / JavaScript (Vanilla)
- Tailwind CSS (Giao diện)
- Lucide Icons (Biểu tượng)
- SheetJS (Xuất Excel)
- jsPDF (Xuất PDF)
